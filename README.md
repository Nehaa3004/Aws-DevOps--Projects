# 🚀 Projects

## 1. Serverless Web Application (AWS)

### Tech Stack
AWS Lambda, API Gateway, DynamoDB, S3

### Description
- Built a fully serverless backend using AWS Lambda and API Gateway.
- Developed REST APIs to handle backend logic without server management.
- Used Amazon DynamoDB for NoSQL data storage.
- Hosted static frontend on Amazon S3.
- Implemented IAM roles and policies for secure service-to-service communication.
- Designed a scalable serverless architecture on AWS.

## 2. CI/CD Pipeline for 3-Tier Application Deployment

### Tech Stack
GitHub Actions, Docker, AWS EC2, Nginx

### Description
- Designed and implemented a CI/CD pipeline using GitHub Actions.
- Automated build and deployment process on every code push.
- Containerized application using Docker.
- Deployed application on AWS EC2 instance.
- Configured Nginx as a reverse proxy for routing traffic.
- Managed environment variables and secrets using GitHub Secrets.
- Reduced manual deployment effort through full automation.

## 3. Highly Available Web Application on AWS

### Tech Stack
AWS EC2, Application Load Balancer, Auto Scaling Group, CloudWatch

### Description
- Deployed application on multiple EC2 instances for high availability.
- Configured Application Load Balancer to distribute incoming traffic.
- Implemented Auto Scaling Group to handle variable traffic loads.
- Set up CloudWatch monitoring and alarms for performance tracking.
- Ensured fault tolerance and improved system reliability.
